# MPD Modules #

These files contain a *Reader's release* of textbook resource, *Maize Plant Discipleship*, in various formats. 

## File Formats ##

There are 4 file formats included in this *Readers' Release* of Module 4.

### Non-Editable Files ###

These file are read-only; please record feedback and comments in a plain text file, cross-referencing by page number and, or section heading:

-	ePub — for use with e-readers;
-	html — view in web browsers;
-	pdf — portable document format.

### Editable Files ###

These files may be edited directly; best results will be obtained by using a markdown editor—I recommend [MultiMarkdown Composer][Mmdc], not least because it facilitate [Critic Markup][cm]

-	mmd — plain text source files.

[Mmdc]:	http://multimarkdown.com
[cm]:	http://criticmarkup.com

## Guidelines for Readers

Please check *Readers' Notes* folder for guidance on providing feedback as a Reader.
