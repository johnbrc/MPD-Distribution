#	This folderContains distributable files for a [Maize Plant Discipleship][Further information] handbook. The files are divided into three folders.-	MPD-*.md-	MPD-*tex-	Exports##	MPD-*.md-	Incorporating `MPD-*.md`-	A plain-text, finely-structured document-	Intended to be the **primary source text for translations**The `MPD-*.md` file uses simple [markup] language (called [Multimarkdown]) to define headings, bold, italic, footnotes, tables and to link to included images etc. This [markup] structure and syntax should be maintained precisely by translators, in order to produced a translation with the same structure as the original document.##	MPD-*.tex-	Incorporating `MPD-*.PDF` -	A PDF document, **formatted for print as an A5-size handbook**-	Computer-readable `.tex` files can be can be ignored by most users##	Exports-	HTML, for rendering (structured) source text in a web browser-	E-pub (unsupported, where included)###	Further information-	 [Maize Plant Discipleship project website](http://maizeplantdiscipleship.wordpress.com)-	 [Maize Plant Discipleship digital distribution hub](https://github.com/johnbrc/MPD-Distribution/wiki)[markup]: https://rawgit.com/fletcher/human-markdown-reference/master/index.html[Multimarkdown]: http://fletcherpenney.net/multimarkdown
