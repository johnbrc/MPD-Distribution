#	Maize Plant Discipleship Files**Maize Plant Discipleship** is being prepared for initial publication as a series of low-cost paperback handbooks. To facilitate accessibility—including translation into other languages in particular—[syllabus](syllabus/syllabus.md) modules are being made available digitally: -	 A single, [Editable Source File]-	 A range of [Read-Only Files]### FilenamesDistributed files are named per this convention: 		MPD-Mn, referring to Maize Plant Discipleship syllabus module, n, where *n* is from 1-16, hence, MPD-M4 refers to Module 4###	Editable Source File*Mainly useful to translators*. Written in [MultiMarkdown], they are used to generate [Read-Only Files]. MultiMarkdown is a plain-text language both human- and computer-readable, used to generate LaTex, PDFs, HTML and DOCX files. This README is composed in Multimarkdown, as are many files in this distribution.### Read-Only Files*Useful for review and printing.* PDF and HTML formats.-	PDF: Portable Document File — A5 handbook format; generated using LaTex, a typesetting language, which produces high quality PDFs from plain [Multimarkdown] text. -	HTML: Hyper-Text Markup Language files — with accompanying images; readable in standard web browsers; viewable offline, so highly distributable.## Further Information -	[About](../about/about.md) Maize Plant Discipleship-	Return to [README](../README.md)####	External links-	[LaTex](http://www.latex-project.org)-	[MultiMarkdown Composer](http://multimarkdown.com)[Multimarkdown]: http://fletcherpenney.net/multimarkdown/
