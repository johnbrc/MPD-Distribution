#	Files**Maize Plant Discipleship** is initially being prepared for publication as a series of low-cost paperback handbooks. To facilitate accessibility—including translation into other languages in particular—[syllabus](../about/syllabus.md) modules are being made available digitally: -	 A single [Editable Source File]-	 A range of [Read Only Files]###	Editable Source File*Useful to translators.*-	**MD**: MultiMarkdown[^md] files — used to generate [Read Only Files]. ###	Read Only Files*Useful for review and printing.*-	**PDF**: Portable Document File — A5 handbook format; generated using LaTex[^ltx]-	**HTML**: Hyper-Text Markup Language files and images — readable offline, using standard web browsers.-	Other formats may be added in future, e.g. e-Pub etc.###	FilenamesDistributed files are named per this convention: 		MPD-Mn, referring to Module n, where n is from 1-16 — hence, MPD-M4 = Module 4##	Further Information-	[About](../about/about.md) Maize Plant DiscipleshipReturn to [README](../README.md)[^md]: [Multimarkdown](http://fletcherpenney.net/multimarkdown) is a plain-text language both human- and computer-readable, useful for generating LaTex, PDF, HTML and DOCX files.[^ltx]:  [LaTex](http://www.latex-project.org) is a sophisticated typesetting language, capable of producing high quality PDFs from plain MultiMarkdown text.
