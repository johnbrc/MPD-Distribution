#	Version numbers

Each MPD handbook is identified with a date, time and a version number in: 

-	metadata at the top of editable (MultiMarkdown) source files.
-	a date-time-version stamp located on initial pages of (LaTex-generated) PDFs.

###	Numbering scheme

Version numbers use the following:

| draft 	| phase 	| version number | 
|  :-:	| :-:	| :-:		|  
| first	| draft	| 0.1.x	|
| second	| review 	| 0.2.x	| 
| third	| copy-editing 	| 0.3.x 	| 
| fourth	| translation	| 0.4.x	| 
| fifth	| post-translation	| 0.5.x	| 
| first edition	|  publication	|  1.0 	|
